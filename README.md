
# Proyecto DEV.F



## Fórmula 1

Esta proyecto fue realizo con HTML semantico, y un poco de estilo con CSS dentro de HTML. 
Es una página estilo blog con información de la Fórmula 1, que permite conocer un poco acerca de este deporte como, sus pilotos, circuitos y las escuderias, además del funcionamiento del deporte, es una página diseñada principalmente para principiantes del deporte. 

Esta página contiene muchas imagenes, se adjuntaron en diferentes carpetas, ya que se queria mantener un estilo de tamaño uniforme para tener mejor control de los espacios.
Tamben contiene los links de la redes sociales oficiales de la F1 para que las personas puedan seguir las noticias más actulizadas.

*toda la información fue obtenida de fuentes oficiales, sin embargo esta puede cambiar, ya que estan en proceso el campeonato mundial.
Se agrego una cajita de contacto para enviar información en caso de desactualización de la página.
